ARM-芯片架构简介

Posted on 2022-09-05 Edited on 2022-09-07

简介

ARM(Advanced RISC Machine)架构。是一个精简指令集(RISC,Reduced Instruction Set Computer)处理器架构家族，具有低成本、高性能、低耗电的特性。
ARM架构广泛应用在嵌入式系统，现在也有应用在个人计算机（Apple M1）和超级计算机中。

  • 与ARM同属精简指令集架构的还有RISC-V，MIPS，Xtensa等。
  • 与精简指令集相对应的是复杂指令集（CISC,Complex Instruction Set Computer），代表架构为X86。

ARM架构范畴

ARM架构的主要范畴整理如下：

条目 说明
指令集 每条指令的功能；指令如何在内存中表示(它的编码)
寄存器组 有多少寄存器；寄存器的大小；寄存器的功能；寄存器的初始状态
异常模型 不同的权限级别；异常的类型；从异常中获取或返回时会发生什么
内存模型 内存访问是如何排序的；缓存是如何运作的，什么时候以及软件必须如何执行显式维护
调试、跟踪和分析 如何设置和触发断点；跟踪工具可以捕获什么信息，以什么格式捕获
安全性扩展(TrustZone) 安全性扩展，隔离某些资源
